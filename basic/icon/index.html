图标

提供了 Font Awesome 和 Glyphicons 两套图标集合。

基础用法

  • 示例
  • 代码 
<cb-icon name="edit" />
<cb-icon name="share" />
<cb-icon name="remove" />
<cb-icon name="refresh" spin />
<cb-icon type="g" name="edit" />
<cb-icon type="g" name="share" />
<cb-icon type="g" name="remove" />

属性

参数 说明 类型 可选值 默认值
name 图标名称。具体名称参见图标集合。 string — —
type 图标分类。fa 对应 Font Awesome，g 对应 Glyphicons string fa / g fa
spin 图标是否有旋转动画。 boolean — false
