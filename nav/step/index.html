步骤条

引导用户按照流程完成任务的分步导航条，可根据实际应用场景设定步骤，步骤不得少于 2 步。

基础用法

  • 示例
  • 代码
  • 第一步
  • 第二步
  • 第三步
  • 完成
<cb-steps v-model="step">
  <cb-step>第一步</cb-step>
  <cb-step>第二步</cb-step>
  <cb-step>第三步</cb-step>
  <cb-step>完成</cb-step>
</cb-steps>

属性

参数 说明 类型 可选值 默认值
value 绑定 v-model，设置当前激活步骤 number — 0
