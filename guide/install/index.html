安装

本节将介绍如何安装 Changba UI 组件库。

npm 安装

如果你的项目支持 webpack、vite 等构建工具，可以使用 npm 安装： 

npm install --save changba-ui

CDN 引入

如果你只想从最简单的 HTML 页面开始入手，可以通过 CDN 的方式引入，下面提供了一个参考示例： 

<html>
  <head>
    <!-- 引入样式文件 -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/changba-ui@1.3/lib/theme/index.css">
  </head>

  <body>
    <div id="app"></div>
    <!-- 引入 Vue 和 ChangbaUI 的 JS 文件 -->
    <script src="https://cdn.jsdelivr.net/npm/vue@2.6/dist/vue.min.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/changba-ui@1.3/lib/changba-ui.min.js"></script>

    <script>
      // 在 #app 标签下渲染一个按钮组件
      new Vue({
        el: '#app',
        template: `<cb-button theme="primary">我是按钮</cb-button>`
      });

      ChangbaUI.Notify.notice('提示');
    </script>
  </body>
</html>
